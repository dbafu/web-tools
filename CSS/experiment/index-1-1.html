CSS

CSS stands for Cascading Style Sheet. CSS controls the presentational aspects of your web pages.

Block-Level Elements

Block-Level elements stack down the page. They include:

  • header
  • section
  • h1,h2,ect.

Inline Elements

Inline elements sit next to each other, if there is a room. They include:

  • img
  • a
  • em
